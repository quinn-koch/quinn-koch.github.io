# Quinn Koch — Personal Website

A clean, minimal personal site built for GitHub Pages.

## Files
- `index.html` — Home
- `about.html` — About + Education + Skills
- `experience.html` — Research & Work Experience
- `projects.html` — Research Projects
- `contact.html` — Contact
- `style.css` — Shared stylesheet
- `nav.js` — Mobile nav toggle

## Setup on GitHub Pages

1. Create a new GitHub repository (e.g. `quinn-koch.github.io` or any name).
2. Upload all files in this folder to the root of the repo.
3. Go to **Settings → Pages** and set the source to `main` branch, `/ (root)`.
4. Your site will be live at `https://<username>.github.io/<repo-name>`.

## Adding Your Photo
Replace `photo.jpg` with your actual headshot file named `photo.jpg`.

## Adding Your Resume
Add your resume PDF as `QuinnKoch_Resume.pdf` in the root folder.
The Experience and Contact pages already link to it.

## Customization
- Edit colors in `style.css` under `:root` (look for `--accent`, `--warm`, etc.)
- Add a GitHub link in the navbar or contact page by adding:
  `<a href="https://github.com/YOUR_USERNAME" target="_blank" class="about-link">⌥ GitHub</a>`
